Aufgabe 3:

  • Aufgabe 3: Eine html-Datei beginnt überlicherweise mit folgender Zeile:

    !DOCTYPE HTML PUCLIC "-//W3C//DTD HTML 4.01//EN"

    Beantworte dazu folgende Fragen:

    • Warum muss die erste Zeile einer html-Datei so aussehen?
    • Was ist das W3C?
    • Wie kannst Du prüfen ob Deine Webseite den Regeln des W3C entspricht?
  • Die erste Zeile stellt sicher, dass die HTML-Datei von jedem Browser gleich gerendert wird.
    • In diesem Fall handelt es sich um HTML Version 5.
  • Das W3C ( World Wide Web Consortium ) hat als Ziel die Standardisierung der Techniken im World Wide Web.
    • Gründung: 1. Oktober 1994
    • Vorsitzender: Tim Berners-Lee
    • Es handelt sich um eine Mitgliedsorganisation.
    • Es entwickelt technische Spezifikationen und Richtlinien.
  • Man kann auf der Seite Validator.w3.org überprüfen ob eine Webseite konform ist.
